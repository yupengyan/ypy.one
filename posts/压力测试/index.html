方法一：WRK#

git clone https://github.com/wg/wrk.git  
cd wrk  
make

Example：

./wrk -t12 -c400 -d30s http://127.0.0.1:8080/jartto

参数说明：
-c：HTTP 连接数，每一个线程处理 N = 连接数/线程数
-d：持续时间，2s，2m，2h
-t：总的线程数
-s：脚本，可以是 Lua 脚本
-H：增加 HTTP header，例如：User-Agent: jartto
--latency：输出时间统计的细节
--timeout：超时时间

比较三个应用： ./wrk -t12 -c400 -d30s http://172.19.13.51:8082/annualdata/user/CasLogin ./wrk -t12 -c400 -d30s http://172.19.13.47:8080/annualdata/user/CasLogin ./wrk -t12 -c400 -d30s http://172.19.13.41:8086/annualdata/user/CasLogin ./wrk -t12 -c400 -d30s https://lib.gdufe.edu.cn/annualdata/user/CasLogin
